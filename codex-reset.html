Codex Reset

A readable browser companion for Codex reset credits and manual Codex account handoffs. The page helps you prepare commands, copy operational notes, and keep the flow tidy.

This page is static. It does not call OpenAI endpoints, does not read `auth.json`, does not read cookies, does not use local storage, and does not make network requests. The CLI remains the tool that reads `auth.json` and redeems banked reset credits.

Manual flow

  1. Use the CLI to inspect reset credits before taking action.
  2. Use the official Codex or ChatGPT invite UI when an invite email needs to be sent.
  3. Send only if the official UI says the account is eligible and the action is available.
  4. After the invite is accepted and a first Codex message is sent, use `codex-reset` to spend a banked reset when needed.

Audit notes

  • The page never sends invites or redeems credits.
  • The page never reads `auth.json`, cookies, tokens, or account data.
  • The generated commands are visible before you copy them.
  • Redeeming a reset credit is irreversible once the CLI POST succeeds.
